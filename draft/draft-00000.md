# Add Git Alias to `.gitconfig` ![status:draft]

### Prerequisite
Assume the user know how to use:
- Vim
- Power Shell
- VSCode
- winget

### Quick Start
Run
```powershell
git config edit --global
```
add the following lines:
```
[alias]
        co = checkout
        br = branch
        ci = commit
        st = status
```

### Troubleshooting
**Issue 1**

When you Run
```pwsh
git config edit --global
```
Shows the error: ![status:draft]
```
cannot find the command edit
```

Run:
```pwsh
git --version
```
Expected output:
```
git version 2.46.0.windows.1
```
If the version is <2.46:
```pwsh
# Uninstall the current git from Windows Installer first
winget install git.git
```
You should be able to update the git version.
